# asdf-xnc

[asdf](https://asdf-vm.com) plugin for [xnc](https://github.com/huru-io/xnullclaw) — AI agent orchestration CLI.

## Install

```bash
asdf plugin add xnc https://github.com/huru-io/asdf-xnc.git
asdf install xnc latest
asdf set xnc latest
```

## Use

```bash
# List all available versions
asdf list all xnc

# Install a specific version
asdf install xnc 0.1.0

# Set version for current project
asdf set xnc 0.1.0

# Set global default
asdf set --home xnc latest
```

Check [asdf](https://github.com/asdf-vm/asdf) for more instructions on how to install and manage versions.
